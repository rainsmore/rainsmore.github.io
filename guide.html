📘 Guide : Ajouter un fichier PDF dans le dossier articles et créer son lien

✅ Étape 1 : Aller dans le dépôt GitHub

  1. Va sur le dépôt du site :
    https://github.com/rainsmore/rainsmore.github.io
  2. Clique sur le dossier articles/
    (s’il n’existe pas encore, tu peux le créer en même temps que tu ajoutes un fichier)

⬆️ Étape 2 : Uploader un fichier PDF

  1. Clique sur "Add file" > "Upload files"
  2. Glisse ton fichier PDF (ex : MonArticle.pdf) dans la zone
  3. Vérifie que le chemin est bien articles/MonArticle.pdf
  4. En bas, clique sur "Commit changes" pour valider

🌐 Étape 3 : Créer le lien public GitHub Pages

Le lien public est toujours construit comme ceci :

https://rainsmore.github.io/articles/NOM_DU_FICHIER.pdf

🔁 Exemple :

Si ton fichier s'appelle :
articles/Article Final - Conférence 2025.pdf
L’URL sera :

https://rainsmore.github.io/articles/Article%20Final%20-%20Conf%C3%A9rence%202025.pdf

✅ Remarques :
- Les espaces sont remplacés par %20
- Les caractères spéciaux (é, è, ç…) sont automatiquement encodés

💡 Astuce : Renommer les fichiers

Pour avoir des liens plus simples, évite les espaces et caractères spéciaux :

  • ✅ article_final_conference2025.pdf
  • ❌ Article Final - Conférence 2025.pdf

🔗 Étape 4 : Utiliser le lien dans ton site

Exemple HTML :

<a href="https://rainsmore.github.io/articles/MonArticle.pdf" target="_blank">
  Lire l'article (PDF)
</a>

Remarque : l’attribut target="_blank" ouvre le fichier dans un nouvel onglet.

✅ Résumé rapide

  • 📂 Ajoute ton fichier PDF dans articles/
  • 🔗 Crée l’URL publique avec https://rainsmore.github.io/articles/NOM_DU_FICHIER.pdf
  • 📎 Utilise ce lien sur ton site ou dans un email
